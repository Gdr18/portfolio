Sobre mí

¡Hola! 🖖 Te doy la bienvenida a mi portfolio.

Mi nombre es Gádor y soy una apasionada de la tecnología desde que tengo uso de razón. En un momento de búsqueda personal descubrí el desarrollo web y me enamoré. Siempre he estado en contacto con el arte y la creatividad, ya sea a través del diseño, el lettering, el dibujo o la música. Y siento que el desarrollo me permite combinarlo todo.

A lo largo de mi trayectoria profesional he trabajado en sectores como la logística, el transporte y la hostelería, lo que me ha permitido afrontar diferentes situaciones que me han permitido aprender y crecer.

Pero no todo es trabajo: en mi tiempo libre me encanta hacer ejercicio, disfrutar de mi entorno, jugar a videojuegos y aprender cosas nuevas. Se puede decir que soy una persona activa y curiosa.

Estaría encantada de aportar mis conocimientos y experiencia a tu proyecto. Si buscas a alguien tenaz, creativa y con atención al detalle, ¡no dudes en contactarme!

Proyectos

Ping Server

Este proyecto es un servidor en Python que no solo realiza pings periódicos a una lista de URLs para mantener activos servicios web (ideal para evitar que se duerman en hostings gratuitos), sino que también expone una API ligera desarrollada en Flask que permite gestionar dinámicamente el sistema.

GitHub Postman

La Favorita Bar

API de gestión integral de La Favorita Bar. Incluye autenticación y autorización con JWT y OAuth por roles, envío de emails con SendGrid y gestión de usuarios, productos, pedidos, platos y tokens. Desarrollada con Flask y MongoDB.

GitHub Postman

Calculadora Tkinter

Aplicación de escritorio que permite realizar operaciones matemáticas básicas. Desarrollada en Python con la librería Tkinter, empaquetada con PyInstaller y distribuida mediante instalador creado con Inno Setup.

GitHub

Planet Gamer

E-commerce de videojuegos como proyecto final del "Full Stack Development Certificate".
Incluye frontend en React y backend en Flask, con autenticación de usuarios, gestión de productos, carrito de compras y pagos con Stripe. Utiliza PostgreSQL y SQLAlchemy.

URL GitHub Postman

Redis Link Shortener

Esta app permite generar enlaces cortos. Esta desarrollada en React para el frontend y Flask para el backend. Almacenamiento de datos en Redis.
